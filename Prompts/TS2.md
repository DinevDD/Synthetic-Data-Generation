# TS2 – Data Generation Prompt

> You are a data scientist tasked with generating synthetic data. You are given the XML description of the original dataset. Only base your data generation on the provided inputs, do not search the internet for further description of this or similar datasets, or use any prior knowledge thereof. Generate at least 100 cases. Ensure variety in the generated samples that conform to the distribution in the original dataset. The generated data should be in CSV format with the following header: “Case ID,Activity,Timestamp,Group,Lifecycle”. Ensure to truncate the results by having one row per case and putting the activity workflow in the “Activity” column in this format: “Activity A/Activity B”. The same goes for the “Timestamp”, “Group”, and “Lifecycle” columns, i.e., each is truncated and contains the information for each event.
